01 How to join

  1. Install Prism Launcher

    Grab Prism Launcher (free, open-source, Windows/Mac/Linux). You'll need a Minecraft account and Java 21, which Prism offers to download for you if it's missing.

  2. Download the pack

    Grab the Prism installer (~5 MB), one small file with no account or store needed. Prefer the Modrinth app? Use the .mrpack instead.

  3. Import into Prism

    Add Instance → Import from zip → Browse to the file → OK. It unpacks small, with no mod jars yet; those come on first launch.

  4. Launch once to download mods

    Hit Launch. The first run pauses a minute or two while it fetches every mod (the Prism log shows progress). Later launches are instant. The instance ships pre-tuned at 8 GB min / 12 GB max. Lower the max under Edit → Settings → Java if you have less than 12 GB, but never below 4 GB.

  5. Connect

    Multiplayer → Add Server, then paste the address:

    mc.ishimura.xyz

What's inside

It started as a Create Aeronautics pack and grew into a full kitchen sink: engineering, magic, monsters and a real player economy sharing one world.

Create & engineering

The rotational-engineering core, plus Create Aeronautics (airships and planes), Big Cannons, Create Nuclear, trains via Steam 'n' Rails and dozens of Create add-ons for factories, logistics and jetpacks.

Magic & the occult

Ars Nouveau, Iron's Spells 'n Spellbooks, Occultism, Modular Golems and Grimoire of Gaia. Spellcraft, summoning and rituals running right alongside the gears.

Mobs & bosses

Alex's Mobs, Mowzie's Mobs, L_Ender's Cataclysm, Bosses of Mass Destruction, Mutant Monsters and more. A far busier, far more dangerous world to explore.

Arachnids & horror

Born in Chaos, Rotten Creatures, horror bosses and an arachnophobia expansion, plus blood moons via Enhanced Celestials to make the nights properly grim.

Combat & chaos

Better Combat, TaCZ firearms, Create: Big Cannons and SecurityCraft. Defend your builds, raid the dungeons, and blow plenty of things up.

Economy & colonies

A real player economy: Numismatics currency, Create: Trading Floor and Blockchain, plus MineColonies towns and chatty TownTalk villagers.

Food & farming

Farmer's Delight, the whole Let's Do series (Vinery, Bakery, Meadow…) and Cooking for Blockheads. Proper cuisine, not just steak.

Worldgen & building

Terralith and Tectonic terrain, YUNG's overhauled dungeons, When Dungeons Arise, plus Supplementaries, Macaw's and Rechiseled to build with.

Hundreds of mods and counting. It's pre-alpha and shifts between versions, so the installer always pulls the current set and auto-syncs each launch.

    05 FAQ

    It won't launch or it crashes on world load.

    Most of the time it's too little RAM or a half-finished first download. Make sure Java 21 is installed, give the instance 8 GB, and let the first launch finish fetching mods. Still stuck? Open .minecraft/logs/latest.log, find the Caused by line, and drop the log in the Discord.

    My airship looks broken with shaders on.

    A known upstream bug with Iris/Oculus shaders: they render Aeronautics ships wrong. Turn shaders off when flying.

    A mod misbehaves while I'm on a moving ship.

    The compat layer is meant to handle exactly that. If a specific mod is still wrong on a contraption, mention which one in the Discord so it can be flagged.

    Can I add my own mods?

    Client-only cosmetic mods are usually fine. Anything touching world content or networking will desync you from the server, so ask first.

    How do I report a bug or request a mod?

    Post in the Discord. That's the fastest way to reach the people who run the pack, and you don't need an account anywhere else.
