01 How to join

  1. Install Prism Launcher

    Grab Prism Launcher (free, open-source, Windows/Mac/Linux). You'll need a Minecraft account and Java 21 — Prism offers to download Java for you if it's missing.

  2. Download the pack

    Grab the Prism installer (~5 MB) — one small file, no account or store needed. Prefer the Modrinth app? Use the .mrpack instead.

  3. Import into Prism

    Add Instance → Import from zip → Browse to the file → OK. It unpacks small — no mod jars yet, those come on first launch.

  4. Launch once to download mods

    Hit Launch. The first run pauses a minute or two while it fetches every mod (the Prism log shows progress). Later launches are instant. The instance ships pre-tuned at 8 GB min / 12 GB max — lower the max under Edit → Settings → Java if you have less than 12 GB, but never below 4 GB.

  5. Connect

    Multiplayer → Add Server, then paste the address:

    mc.ishimura.xyz

What's inside

It started as just Create Aeronautics. It did not stay that way. Roughly what's in there now:

Create & engineering

The Create core, plus Aeronautics for airships and planes, Big Cannons, Create Nuclear, trains from Steam 'n' Rails, and a silly number of add-ons for factories, logistics and jetpacks.

Magic & the occult

Ars Nouveau, Iron's Spells 'n Spellbooks, Occultism, Modular Golems, Grimoire of Gaia. Spells, summoning and rituals if you'd rather not touch a gearbox.

Mobs & bosses

Alex's Mobs, Mowzie's Mobs, Cataclysm, Bosses of Mass Destruction, Mutant Monsters. The world's a lot busier, and a lot more likely to kill you.

Arachnids & horror

Born in Chaos, Rotten Creatures, horror bosses, and yes, an arachnophobia expansion. Blood moons from Enhanced Celestials make some nights genuinely rough.

Combat & chaos

Better Combat, TaCZ guns, Big Cannons and SecurityCraft. For raiding dungeons, defending your base, and generally causing problems.

Economy & colonies

An actual economy: Numismatics currency, Create's Trading Floor and Blockchain, plus MineColonies towns and TownTalk villagers.

Food & farming

Farmer's Delight and the whole Let's Do lineup (Vinery, Bakery, Meadow…), plus Cooking for Blockheads. You can cook real food instead of inhaling steak.

Worldgen & building

Terralith and Tectonic for terrain, YUNG's dungeons, When Dungeons Arise, and Supplementaries / Macaw's / Rechiseled for building.

There are hundreds of mods and the list changes all the time — it's pre-alpha. The installer just pulls whatever's current each launch, so you don't have to keep up.

05 FAQ

It won't launch or it crashes on world load.

Most of the time it's too little RAM or a half-finished first download. Make sure Java 21 is installed, give the instance 8 GB, and let the first launch finish fetching mods. Still stuck? Open .minecraft/logs/latest.log, find the Caused by line, and drop the log in the Discord.

My airship looks broken with shaders on.

It's a known bug with Iris/Oculus shaders, nothing we can fix on our end. Just turn shaders off when you're flying.

A mod misbehaves while I'm on a moving ship.

The compat layer is meant to handle exactly that. If a specific mod is still wrong on a contraption, mention which one in the Discord so it can be flagged.

Can I add my own mods?

Client-only cosmetic mods are usually fine. Anything touching world content or networking will desync you from the server — ask first.

How do I report a bug or request a mod?

Post it in Discord. That's where everyone who runs the pack actually is, and you don't need an account anywhere else.
