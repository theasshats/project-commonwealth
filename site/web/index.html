01 How to join

  1. Install Prism Launcher

    Grab Prism Launcher (free, open-source, Windows/Mac/Linux). You'll need a Minecraft account and Java 21 — Prism offers to download Java for you if it's missing.

  2. Download the pack

    From the latest release, grab derpack-x-prism-installer-X.Y.Z.zip (~5 MB, recommended). Modrinth-app users can take the .mrpack instead.

  3. Import into Prism

    Add Instance → Import from zip → Browse to the file → OK. It unpacks small — no mod jars yet, those come on first launch.

  4. Launch once to download mods

    Hit Launch. The first run pauses a minute or two while it fetches every mod (the Prism log shows progress). Later launches are instant. The instance ships pre-tuned at 8 GB min / 12 GB max — lower the max under Edit → Settings → Java if you have less than 12 GB, but never below 4 GB.

  5. Connect

    Multiplayer → Add Server, then paste the address:

    mc.ishimura.xyz

Highlights

  • Create Aeronautics — physics-simulated airships, planes & vehicles built from Create contraptions. The reason the pack exists.
  • Create & add-ons — the rotational-engineering core everything else is built around.
  • A fuller world — Alex's Mobs and friends make the overworld worth flying over.
  • It just works on ships — a compat layer patches third-party mods (sleeping, chains, mobs, turrets…) to behave on moving contraptions.
  • Quality of life — EMI recipe viewer, AFK sleep, and sane defaults so you build instead of fighting menus.

Pre-alpha and changing between versions — the launcher always installs the current set, and the installer instance auto-syncs each launch.

05 FAQ

It won't launch or it crashes on world load.

Most of the time it's too little RAM or a half-finished first download. Make sure Java 21 is installed, give the instance 8 GB, and let the first launch finish fetching mods. Still stuck? Open .minecraft/logs/latest.log, find the Caused by line, and drop the log in the Discord.

My airship looks broken with shaders on.

Known upstream bug — Iris/Oculus shaders render Aeronautics ships wrong. Turn shaders off when flying.

A mod misbehaves while I'm on a moving ship.

The compat layer is meant to handle exactly that. If a specific mod is still wrong on a contraption, mention which one in the Discord so it can be flagged.

Can I add my own mods?

Client-only cosmetic mods are usually fine. Anything touching world content or networking will desync you from the server — ask first.

How do I report a bug or request a mod?

Post in the Discord. That's the fastest way to reach the people who run the pack — no account on anything else required.
