# `shaderpacks/` — Bundled shaderpacks

Drop `.zip` shaderpacks here. Available to users in Iris/Oculus shader settings.

> ⚠️ **Aeronautics has known visual issues with shaders.** Airships and physics contraptions render incorrectly when shaders are enabled. The mod team is aware. Until they fix it, users may want to disable shaders when flying.

## How to add one

1. From your version-named branch (e.g. `0.3.3`), drop the `.zip` here, e.g. `shaderpacks/complementary.zip`.
2. Run any editor operation to trigger an index refresh.
3. Commit, push, open a PR.

Make sure the shaderpack is compatible with the Iris/Oculus version we ship (currently Oculus on NeoForge 1.21.1). Some shaders only work with newer versions of Iris.
